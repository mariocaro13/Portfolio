NaPlanner

NaPlanner is a native android application developed in java.

Luis Manuel Borges and I developed it together as a final degree work, the application consists of a task manager, which has two types of users, teachers and students.

Teachers can assign tasks to their students, edit them and see if they have completed them. Students can mark them as completed. In addition, all users can generate their own tasks. Tasks are classified in three types, legendary, epic and normal and each one is assigned with a corresponding color. Also depending on the type of user application has a color palette or another, teachers have a blue palette while students implement a palette in green tones.

This application implements design patterns, MVVM, and a firebase database.
